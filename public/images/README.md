Plaats hier je eigen afbeeldingen om de placeholders in de site te vervangen.

Gebruik de volgende bestandsnamen (jpg/png) voor directe vervanging van de gebruikte afbeeldingen in `data.ts`:

- blueprint.jpg
- living_room.jpg
- stairs.jpg
- window_view.jpg
- wood_detail.jpg
- facade_modern.jpg
- portrait.jpg
- villa_forest.jpg
- minimalist_glass.jpg
- interior_luxe.jpg

Voor projecten: de code probeert eerst `/images/<key>.jpg` te laden (bijv. `/images/facade_modern.jpg`). Als een lokaal bestand ontbreekt, valt de app automatisch terug op de externe Unsplash-afbeelding.

Tips:
- Gebruik vergelijkbare resolutie (minimaal 1200px breed) voor goede weergave.
- Geef bestanden korte, ASCII-gebaseerde namen zoals hierboven.
